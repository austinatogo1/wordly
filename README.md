# Wordly — Interactive Dictionary

Wordly is a modern, single-page web app that lets you search for word definitions, pronunciations, synonyms, antonyms, and more, all in one place. Powered by the [Free Dictionary API](https://dictionaryapi.dev/), Wordly is fast, accessible, and works entirely in your browser.

## Features

- **Search for any English word** and get definitions, phonetics, and usage examples
- **Listen to pronunciations** with a single click
- **View synonyms and antonyms** for richer vocabulary
- **Save your favorite words** (stored locally in your browser)
- **Light/Dark theme toggle** for comfortable reading
- **Responsive design** for desktop and mobile
- **Accessible UI** with keyboard navigation and ARIA labels

## Getting Started

1. **Clone or download this repository**
2. Open `index.html` in your browser — no build step or server required!

```
wordly/
├── index.html
├── css/
│   └── style.css
└── js/
    └── app.js
```

## Usage
- Type a word in the search box and press Enter or click "Search →"
- Click the star (☆) to save a word; saved words appear below for quick access
- Click the speaker icon to hear pronunciation (if available)
- Click on synonyms or antonyms to search those words instantly
- Use the theme toggle in the header to switch between light and dark mode

## Tech Stack
- **HTML5**
- **CSS3** (with custom properties and responsive design)
- **Vanilla JavaScript** (no frameworks)
- **[Free Dictionary API](https://dictionaryapi.dev/)**
